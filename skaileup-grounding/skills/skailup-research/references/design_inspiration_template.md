# Design Inspiration Template

Use this structure for `_grounding/general/design_inspiration.md`.
This file directly feeds into `brand-visual` and `screens`.

## Research Categories

| Category | What to look for |
|----------|-----------------|
| **Layout patterns** | Common page structures for this app type (dashboard layouts, list/detail, wizard flows, kanban boards) |
| **Color approaches** | How competitors and best-in-class apps in this space use color |
| **Typography trends** | Font choices that work for this audience and domain |
| **Component patterns** | Navigation styles, data display patterns, form layouts |
| **Onboarding flows** | How similar apps handle first-time users |
| **Empty states** | How similar apps handle zero-data scenarios |
| **Brand references** | Apps outside the domain with a visual feel that could work |

When finding reference URLs, capture screenshots via the browser tool where possible.
Save to `_grounding/findings/`.
