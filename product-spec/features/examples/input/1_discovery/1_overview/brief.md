# TaskFlow

## Vision
A lightweight task management app for small teams. No enterprise bloat —
just tasks, assignments, and progress tracking.

## Target Audience
Small teams (2-10 people) in startups, freelancers, and small agencies
who find tools like Jira too heavy but need more than a shared spreadsheet.

## Core Problem
Small teams waste time managing their tool instead of managing their work.
They need something that takes 5 minutes to set up and stays out of the way.

## Hero Flow
1. User creates a task with a title and optional description
2. Assigns it to a teammate
3. Teammate sees it in their dashboard
4. Task moves through statuses: todo → in progress → done
5. Team sees a simple overview of what's happening
