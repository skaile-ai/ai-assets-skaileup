# Design Evaluation Rubrics

Referenced by the eval-product skill. Score each dimension 0–10. Be strict — mediocre work scores 5–6, not 7–8.

## Quality of Design (0–10)

Is the UI a coherent experience or separate components strung together?

| Score | Description |
|-------|-------------|
| 9–10 | Every screen intentionally designed as part of a whole. Visual hierarchy guides attention. Navigation feels natural. |
| 7–8 | Generally coherent with minor inconsistencies. |
| 5–6 | Some coherent sections, noticeable disconnects (e.g. settings looks like a different product). |
| 3–4 | Mostly disconnected. Different sections feel like different apps. |
| 0–2 | No coherence. Random layout decisions throughout. |

## Originality (0–10)

Does this look like a real product with a distinctive identity, or a generic AI output?

AI tends to converge on the same patterns. Penalise these defaults:
- Purple/blue gradient as primary brand color
- Centered hero section with large headline + subtext + CTA button
- Excessive card-based layouts for all content types
- Default Tailwind gray palette with one accent color
- Generic sans-serif (Inter, Roboto) without typographic personality
- "Clean and minimal" as a design goal rather than a visual choice

| Score | Description |
|-------|-------------|
| 9–10 | Distinctive visual identity. You would recognize this product's brand in a screenshot. |
| 7–8 | Some distinctive choices, residual AI defaults in secondary areas. |
| 5–6 | Mostly default patterns. One or two custom choices. |
| 3–4 | Heavy AI defaults. Purple gradients, card grids, generic hero. |
| 0–2 | Indistinguishable from AI-generated placeholder UI. |

## Craft (0–10)

Are the details right? Typography, spacing, color harmony, consistency.

| Score | Description |
|-------|-------------|
| 9–10 | Every detail considered. Font pairing works. Spacing consistent and rhythmic. No rough edges. |
| 7–8 | Mostly polished, a few rough edges. |
| 5–6 | Functional but not polished. Inconsistent font sizes or uneven spacing in places. |
| 3–4 | Noticeable inconsistencies throughout. |
| 0–2 | No apparent attention to craft. |

## Functionality (0–10)

Does every UI element serve a clear user purpose?

| Score | Description |
|-------|-------------|
| 9–10 | Every element earns its place. No decorative clutter. Interactions are intuitive. |
| 7–8 | Mostly purposeful with minor gratuitous elements. |
| 5–6 | Some components exist for aesthetics without function. |
| 3–4 | Significant decorative-over-functional imbalance. |
| 0–2 | UI elements that actively harm usability. |
