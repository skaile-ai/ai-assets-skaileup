## What lives here

The **skaileup catalog** is the skill library consumed by the `skaile` CLI and the
agent runtime. ~94 SKILL.md files organized into 17 domains across three groups,
plus the flow + bundle pairs that compose them into per-tier pipelines.

<CardGrid>
  <Card title="Concept" icon="document">

## How to read this site

1. **Start with [What is Skaileup](/intro/what-is-skaileup/)** if you've never
   touched the catalog before.
2. **The [Mental Model](/intro/mental-model/)** explains the two-group structure,
   slice loops, and tier system in one page.
3. **Each domain** has an overview page plus one page per skill — every skill
   page embeds the verbatim `SKILL.md` body and links to the source on GitHub.
4. **The [Reference](/reference/skill-grammar/)** section captures the DSL
   grammar, frontmatter schemas, and the iron laws every skill obeys.
