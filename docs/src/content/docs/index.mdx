## What lives here

The **skaileup collection** is the skill library consumed by the `skaile` CLI and the
agent runtime. ~94 SKILL.md files organized into 17 domains across three groups,
plus the self-contained flows that compose them into per-tier pipelines.

## How to read this site

1. **Start with [What is Skaileup](/intro/what-is-skaileup/)** if you've never
   touched the collection before.
2. **The [Mental Model](/intro/mental-model/)** explains the two-group structure,
   slice loops, and tier system in one page.
3. **Each domain** has an overview page plus one page per skill — every skill
   page embeds the verbatim `SKILL.md` body and links to the source on GitHub.
4. **The [Reference](/reference/skill-grammar/)** section captures the DSL
   grammar, frontmatter schemas, and the iron laws every skill obeys.

## Status

The collection reorganization is complete (Phases 0–3). Phase 1 (two-group
restructure, ~70 skills moved) shipped 2026-05-07; Phase 2 (quality remediation —
frontmatter, DOMAIN.md authoring, validators, CI gates) completed 2026-05;
Phase 3 (SKILL_GRAPH migration — all 11 deferred tier-flow skills built,
`deferred_skills.yaml` now empty) completed 2026-05-30. See the
[Roadmap](/improvements/) for the full phase record and remaining non-blocking
follow-ups.
