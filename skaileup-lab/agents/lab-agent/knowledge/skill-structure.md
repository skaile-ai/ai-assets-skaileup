# Skill Structure

Every skill lives in its own directory:

```
my-skill/
├── SKILL.md              ← Required. YAML frontmatter + markdown prompt
├── recipes/              ← Step-by-step implementation instructions
│   ├── setup.md
│   └── feature-x.md
├── examples/             ← Atomic code examples
│   └── use-fetch.md
├── scripts/              ← Validation scripts
├── test-manifest.yaml    ← Lab test definitions
├── tests/cases/          ← Individual test case definitions
├── learnings/            ← Auto-generated by lab-learn
│   ├── corrections.md
│   ├── learnings.md
│   └── candidates.md
└── references/
    └── versions.json     ← Framework version tracking
```

The SKILL.md frontmatter contains: name, description, source, version, keywords, user_inputs, reads_from, writes_to.
